###电子商务网站静态页面



####任务描述
一个电商网站的整站页面布局开发，一共实现了九个页面的布局，也就是常见的电商网站的基本布局。
####分页面有：
- 首页
- 登陆页
- 注册页
- 产品介绍页面
- 产品分类页面
- 商品介绍页面
- 商品评价页面
- 商品筛选页面
- 购物车结算页面
